The dataset that we choose was regarding the total number of covid-19 cases and deaths in countries all around the world, however to be able to not overcrowd the graphs we choose the top 40 highest covid-19 cases. The dataset provides details regarding the country, covid-19 cases, covid-19 deaths and region specific to the country. Countries where COVID-19 has spread
2. The bar chart is an appropriate visual encoding for the first visualization because the x-value which is country is a categorical data and the y-value for covid-19 cases are quantitative, so it makes sense to create a barchart to illustrate this. However, while the second visualization is a line chart and shouldn't have used an categorical value for the x-axis, we felt that showing both the covid-19 death cases and covid-19 cases on this dataset provided an insightful outlook that the death resulting from the virus wasn't that significant.
3. The colormap that we used for the two vizualization are static, because we wanted our viewers to have a clean and simple design that reveals the stark impact covid-19 had on our world.
4. The designs that we choose are mostly simple designs as we wanted to provide overview first of the impact covid-19 had on these 40 countries. We choose the color red for the cases since red is closely related to danger, which symbolizes how dangerous Covid-19 was. On the second visuzalition, we choose the color black to represent the death caused by the Coronavirus, because black is the color that we normally wear when mourn the dead.
